# Claude Code 新手指南

一份面向零基础用户的 Claude Code 中文入门网页。内容根据 Anthropic 官方文档整理，覆盖安装登录、代理工作方式、任务表达、常见工作流、权限安全、`CLAUDE.md`、会话管理和命令速查。

## 在线阅读

<https://aklycoh.github.io/claude-code-beginner-guide/>

## 本地预览

```bash
npm run dev
```

打开 <http://127.0.0.1:4173/>。

## 构建

```bash
npm run build
```

静态文件将生成到 `dist/`。推送到 `main` 后，GitHub Actions 会自动部署 GitHub Pages。
